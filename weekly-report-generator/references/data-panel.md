# 数据面板自定义模板

行政周报默认数据面板。在对话中说出"主要看XXX和YYY"即可自定义。

## 默认面板（行政通用）

```
文件处理 X件 | 会议组织 X场 | 检查接待 X次 | 设施维护 X项
```

## 自定义面板示例

### 偏文书管理
```
文件处理 X件（发文Y件/收文Z件）| 会议纪要 X份 | 印章使用 X次
```

### 偏后勤保障
```
设施报修 X项（已完成Y项）| 采购 X项（已完成Y项）| 物业协调 X次
```

### 偏对外联络
```
检查接待 X次 | 发函 X件 | 外部协调 X项
```

### 偏综合管理
```
文件处理 X件 | 会议组织 X场 | 培训学习 X次 | 制度修订 X项
```

## 使用方式

用户在对话中说出偏好，如"加上培训人次和满意度"，后续报告自动添加该指标。

设定后，同一会话内所有后续报告沿用自定义面板。

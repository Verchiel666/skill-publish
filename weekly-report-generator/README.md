# 行政周报生成 Skill

**Version**: V3.0 | **Author**: Verchiel | **License**: MIT

---

## 概述

专为行政/文秘/基层管理者设计的结构化工作汇报生成器。把一周零散的工作记录自动转化为格式规范、逻辑清晰、可直接提交的工作报告。

## 核心卖点

| 功能 | 说明 |
|------|------|
| 📊 四档出稿 | 简报/Word即贴/IM群发/详尽(含时间轴) |
| 🔄 跨期追踪 | 上周计划自动对照完成率，未完成自动承接 |
| 📈 进度可视化 | 多项目独立进度条，滞后自动预警 |
| ⏱️ 时间轴视图 | 详尽模式按日排列本周里程碑 |
| 🎨 表达引擎 | 每周自动轮换句式，告别流水账 |
| 🔗 关联逻辑 | 事项间因果/协同关系自动标注 |
| 📋 自定义面板 | 说出关注的指标，后续自动沿用 |
| 💬 IM即贴 | 群发版本，纯文本+emoji，手机3屏 |
| 📄 Word即贴 | 复制粘贴零排版 |

## 适用人群

- 行政/文秘/办公室人员
- 基层管理者
- 任何每周需要写工作汇报的人

## 工作自动归类（10大板块）

# Demand Detective（需求侦探）

**Version**: 1.0.0 | **Author**: Verchiel | **License**: MIT

---

## 概述

把模糊需求变成可执行、可共识、可追踪的开发方案。通过场景反讲、A/B 选择题和决策日志，将"一句话需求"转化为分级开发任务单。

## 核心卖点

| 功能 | 说明 |
|------|------|
| 🔍 场景反讲 | 先确认业务故事，不急着给方案 |
| ❓ A/B 选择题 | 帮用户理清模糊点，降低决策成本 |
| 📋 决策日志 | 记录关键选择，回溯不迷路 |
| 📦 分级任务单 | 输出可执行的开发任务，含优先级 |

## 触发关键词

新需求、提个需求、帮忙设计、优化一下、做个功能、用户反馈、能不能、想要一个、流程、报表、功能、模块

## 工作流

三阶段：场景共建 → 方案选择 → 任务输出

## 适用人群

- 产品经理
- 技术负责人
- 需要对接开发团队的业务人员

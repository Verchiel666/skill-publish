# 行政公文起草 Skill

**Version**: V5.0 | **Author**: Verchiel | **License**: MIT

---

## 概述

专为行政/文秘/办公室人员设计，覆盖**通知、请示、函件、纪要等 15+ 种公文类型**的起草→改写→批量→修正全流程。

## 核心卖点

| 功能 | 说明 |
|------|------|
| 🚀 三档出稿 | 精简(群里发) / 标准(日常工作) / 详尽(对外红头) |
| 📄 Word即贴 | 复制粘贴到WPS/Word，零二次排版 |
| 💬 对话修改 | "改正式一点""时间改成周四""不是通知是函件" 一句话搞定 |
| 👥 多人纪要 | 散乱口述自动按发言人分类整理 |
| 📦 批量生成 | 同一通知，一键生成给N个部门的版本 |
| ✍️ 改写草稿 | 把你写的口语草稿直接改成正式公文 |
| 🔍 歧义澄清 | "报告"是汇报还是情况报告？主动确认不出错 |
| 📝 7套模板 | 通知/请示/纪要/函件/讲话稿/总结/制度 |

## 适用人群

- 行政/文秘/办公室人员
- 基层管理者（需要写请示、报告）
- 任何每周需要写 3+ 份公文的人

## 覆盖公文类型（15种）

通知 | 请示 | 报告 | 函件 | 纪要 | 通报 | 批复 | 决议 | 公告 | 通告 | 意见 | 制度/办法 | 工作计划 | 工作总结 | 讲话稿/致辞

## 使用示例

### 快速通知
> 速稿：周五下午2:30消防演练，全员参加

### 正式请示
> 写个请示，申请采购一台高速扫描仪，预算5000，现在的用了八年老卡纸

### 改写草稿
> 帮我把这段话改成正式通知："各位，楼下停车场下周一开始划线，大家提前把车挪走"

### 批量生成
> 给财务部、人事部、业务部各发一个通知，下周三下午开预算会

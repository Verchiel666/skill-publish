# xparse-parse

**Version**: 1.0.0 | **Author**: Verchiel | **License**: MIT

---

## 概述

通过 xparse-cli 将文档解析为干净的 markdown 或结构化 JSON。支持 PDF、图片、Office 文件、HTML、OFD 等格式。处理加密 PDF、页面范围、markdown/文本输出及详细结构化提取。

## 核心卖点

| 功能 | 说明 |
|------|------|
| 📄 多格式支持 | PDF、图片、Doc(x)、Ppt(x)、Xls(x)、HTML、OFD、RTF 等 |
| 🔓 加密 PDF | 自动处理密码保护的 PDF |
| 📊 结构化输出 | Markdown 或 JSON 两种模式 |
| 📍 页面范围 | 支持指定页码范围提取 |
| 🆓 免费可用 | PDF 和图片零配置即可用 |

## 安装

```bash
# 安装 CLI
xparse-cli version
```

## 适用场景

- 把 PDF/图片/Office 文档转成 Agent 可用的 markdown
- 文档批量处理自动化
- 加密 PDF 的内容提取
